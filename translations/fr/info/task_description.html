En gardant à l'esprit vos connaissances sur les variables et les opérations qu'elles permettent d'effectuer, faisons un pas de côté et parlons des fonctions. Une fonction est un bloc de code qui peut être utilisé plusieurs fois à différents endroits. En général, une fonction reçoit des données d'entrée (paramètres), effectue des opérations avec elles et renvoie une valeur à l'extérieur.

1. Dans cette mission, vous devez définir une fonction func, qui ne prend aucun argument.

2. Votre fonction ne fera rien non plus! Pour qu'un code (expression) fonctionne correctement sans rien faire, vous devez utiliser ce que l'on appelle des placeholders: le mot-clé pass ou l'opérateur ....

3. Notez que même si une fonction ne renvoie rien, elle renvoie de toute façon la valeur vide None.
