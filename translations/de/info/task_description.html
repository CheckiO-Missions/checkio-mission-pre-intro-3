Nachdem Sie sich Ihr Wissen über Variablen und Operationen mit ihnen gemerkt haben, lassen Sie uns einen Schritt zur Seite machen und über Funktionen sprechen. Eine Funktion ist ein Codeblock, der mehrfach an verschiedenen Stellen verwendet werden kann. Normalerweise erhält eine Funktion einige Eingabedaten (Parameter), führt einige Operationen mit ihnen durch und gibt einen Wert zurück.

1. In dieser Aufgabe müssen Sie eine Funktion func definieren, die keine Argumente benötigt.

2. Ihre Funktion wird auch nichts tun! Damit ein Code (Ausdruck) korrekt funktioniert, ohne etwas zu tun, müssen Sie so genannte Platzhalter verwenden: das Schlüsselwort pass oder den Operator ....

3. Beachten Sie, dass eine Funktion, auch wenn sie nichts zurückgibt, trotzdem den leeren Wert None zurückgibt.
